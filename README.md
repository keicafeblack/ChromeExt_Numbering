# **透過ナンバリング (Transparent Numbering)**

Web会議や画面共有時の「あ、その右のボタン…いや、もうちょっと上…」という指示出しのストレスを解消するためのChrome拡張機能です。

画面上の入力フィールドやボタンを自動で検出し、**見た目の順序（左上から右下）** に基づいて連番（1, 2, 3...）をオーバーレイ表示します。

*(ここに実際の動作スクリーンショットを貼ってください)*

## **✨ 特徴**

* **自動ナンバリング**: ページ内の input, button, a タグなどを自動検出し、番号を振ります。  
* **直感的な順序**: HTMLの記述順ではなく、画面上の座標（左上優先）に基づいて番号を振るため、人間が見たままの順序で番号がつきます。  
* **操作を邪魔しない**: オーバーレイ表示される番号はマウスクリックを透過する（pointer-events: none）ため、番号が表示されたままでも下のボタンを押したり入力したりできます。  
* **カスタマイズ可能**: 設定画面から、番号ラベルの「サイズ」「透明度」「色」を自由に変更できます。

## **📦 インストール方法**

Chromeウェブストアには公開していないため、以下の手順でブラウザに読み込ませてください。

1. このリポジトリをZIPでダウンロードし、解凍します（または git clone します）。  
2. Chromeブラウザを開き、アドレスバーに chrome://extensions/ と入力して移動します。  
3. 画面右上の **「デベロッパーモード」** をONにします。  
4. **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。  
5. 解凍したフォルダ（manifest.json が入っているフォルダ）を選択します。

**注意**: アイコン画像が表示されない場合は、images フォルダを作成し、icon16.png, icon48.png, icon128.png を配置してください。

## **🚀 使い方**

1. **番号を表示する**  
   * Chromeのツールバーにある拡張機能アイコン（「透過ナンバリング」）をクリックします。  
   * ページ上の要素に番号が表示されます。  
   * もう一度アイコンをクリックすると非表示になります。  
2. **設定を変更する**  
   * 拡張機能アイコンを右クリックし、「オプション」を選択します（または拡張機能管理画面から「拡張機能のオプション」をクリック）。  
   * **ラベルの大きさ**、**濃さ（透明度）**、**背景色** を調整し、「Save」ボタンを押すと設定が保存されます。  
   * 表示中の番号にもリアルタイムで反映されます（再表示時）。

## **🤝 開発背景**

Web会議でのシステム操作指示において、指示語（あれ、それ、右のほう）での伝達に限界を感じ、Windowsの音声操作機能（ボイスアクセス）のナンバリング表示にヒントを得て開発しました。  
全てのコードは生成AI (Google Gemini) との対話のみで作成されています。
